v.128: shrink emoji glyphs + Pinnacle bar fold-in + shorter Deck 3 labels

Reported clutter in Decks 1-4 zoom:

1) Emojis were rendering visually huge.  Apple Color Emoji / Segoe UI
   Emoji include extra glyph metrics so an 11px font renders like 14-
   16px text on screen.  Shrunk to:
     .emoji-glyph     11px -> 8px
     .emoji-glyph-sm   9px -> 7px
   Applied in both inline critical CSS and styles.css.

2) "MEETING RMS" + "ATRIUM SHOPS" labels were touching at the bubble
   boundary.  Shortened to:
     "MEETING RMS"  -> "MEET RMS"
     "ATRIUM SHOPS" -> "SHOPS"
   The Atrium-area emojis (Merabella diamond, boutique bag, camera)
   still convey the venue identities.

3) Pinnacle Bar standalone glyph at x=510 y=298 was bleeding into the
   Atrium bubble (Atrium left edge x=515).  Folded the Bar emoji into
   the Pinnacle Grill bubble alongside the Grill emoji:
     PINNACLE
       🥩  🥂
     specialty
   Single bubble, two sub-glyphs (Grill / Bar), each individually
   tappable and routing to its own venue title.
